# LD-USB
It's a usb to pixel controller. This controller is designed for Jinx and standalone playback.

LD-USB是一個簡單易用的LED Pixel控制器, 為小型商演/Maker/DIY提供另一種選擇.

功能如下:
1. 支援JINX即時輸出顯示
2. 支援Standalone模式, 無須PC即插即播
3. 最多768點輸出
4. 隔離電路設計, 分離USB和驅動訊號
5. 16 MBytes Flash容量
6. DIP switch 多種功能切換
7. 目前只支援ws2812/sk6812/ws2815(800k)
8. mass storage 韌體更新
